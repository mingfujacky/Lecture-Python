# 職場小叮嚀

- 厚臉皮: 該說No就說No, 別硬撐，有問題就問，不懂就做實驗
- 說人話: 讓別人知道你做的事，是有難度有貢獻的，挑重點說白話
- 抬頭望: 別只守著自己的田，看看隔壁鄰居的莊稼如何，有沒有新的做法，也看看有沒有偏離地主的方向，別地主想改種芋頭，而你只會種地瓜
