# 主機板及其住戶
1. CPU插槽
2. 記憶體插槽
3. 北橋晶片: 連接高速週邊，如CPU、記憶體和顯示卡
4. 南橋晶片：連接功能面週邊，如硬碟、USB、網路卡
5. 擴充卡插槽
6. 硬碟/光碟機插槽
7. 週邊接口
8. BIOS及電池
9. 電源供應
