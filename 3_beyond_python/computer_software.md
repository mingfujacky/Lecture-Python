# 電腦的軟體
#### 作業系統 (Operating System, OS)

- 讓使用者能更方便的操作電腦
  *-* 管理檔案系統 & 使用應用程式
  *-* 操作網路 & 控制多媒體裝置
- 使電腦資源得以更有效率的運用
  *-* 管理處理器和記憶體
  *-* 控制輸入與輸出裝置
- 提供程式執行的環境
#### 應用程式 (Applications, App)
Chrome, Anti-virus, Office, Game

2. 列印當前使用者名稱 (whoami)
3. 看看硬碟內有哪些檔案 (dir or ls)
4. 看看有沒有安裝Python軟體 (python –V or python3 –V)
